> **测试日期**: 2026-03-22  
> **测试对象**: 前端公开版（mock/localStorage 运行模式）

## 说明

当前仓库只保留公开版前端的验证方式：

- 页面渲染与交互基于 `mockRuntime` 和浏览器 `localStorage`
- 单元测试、组件测试与 E2E 测试均不依赖私有后端
- 本文档不记录生产环境、安全策略、私有 API 或运维实现

## 本地验证命令

```bash
npm install
npx tsc --noEmit
npm run lint
npm run test:run
npm run test:e2e
npm run build
```
